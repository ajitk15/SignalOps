# Two Claude surfaces, on purpose. `anthropic` (the Client SDK) answers the
# reasoning nodes: one structured JSON reply each, schema enforced by the API.
# `claude-agent-sdk` runs the implementer, which needs an agent loop over file
# tools — the alternative is reimplementing Read, Edit, Glob and Grep and the
# loop around them. Its built-in Bash tool is disallowed; the repository's test
# suite is run by a deterministic node from validated configuration.

claude-agent-sdk>=0.2,<1.0
